Q1:What is the main difference between Dictionary and Non-Dictionary attacks?
A dictionary attack uses, for example, a list of known common passwords/cracked hashes and/or known hashes from the site DB to crack a password by cycling through the wordlist to find matching hashes. A non-dictionary attack either sources the password via other means or simply attempts to bruteforce the correct password.

Q3:What concrete security benefits are achieved by using longer passwords instead of shorter ones?
A longer password exponentially increases the time consumed by a bruteforce attack, making them unviable. A long, complex password makes it so that even if the attacker knows a part the password, they cannot use it to crack the rest via brute force.
